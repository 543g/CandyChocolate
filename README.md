# 🍫 Candy & Chocolate Game Assistant

A quality-of-life enhancement tool for keyboard escape games on Roblox, specifically designed for "Candy & Chocolate" gameplay optimization.

## ✨ Features

### Core Functions
- **🚀 Auto Walk** - Automated forward movement assistance
- **⚡ Speed Adjust** - Customizable walk speed controls (16-150)
- **💤 Idle Prevention** - Automatic activity maintenance
- **🔄 Auto Progress** - Streamlined advancement system
- **👁️ Camera Zoom** - Enhanced field of view (70-120 FOV)

### User Interface
- Modern dark theme with smooth animations
- Draggable interface (drag from header)
- Real-time statistics display
- Smooth toggle switches and sliders
- Clean, intuitive design

## 📦 Installation

### Method 1: Direct Execution
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/543g/CandyChocolate/master/CandyAssist.lua"))()
```

### Method 2: Manual Load
1. Copy the contents of `CandyAssist.lua`
2. Paste into your executor
3. Execute the script

## 🎮 Usage

1. Execute the script in-game
2. The assistant panel will appear on the right side
3. Toggle features on/off as needed
4. Adjust sliders for speed and camera settings
5. Drag the interface from the header to reposition

## ⚙️ Settings

### Auto Walk
Enables automatic forward movement. Your character will move forward periodically.

### Speed Adjust
Allows you to customize your walk speed using the slider (default: 60, range: 16-150).

### Idle Prevention
Prevents being kicked for inactivity during extended play sessions.

### Auto Progress
Automatically triggers progression when reaching speed milestones (1K+ speed).

### Camera Zoom
Increases your field of view for better visibility (adjustable 70-120 FOV).

## 📊 Statistics Display

The assistant shows real-time stats:
- ⚡ Current Speed
- 🔄 Progress/Rebirths
- 🏆 Total Wins
- ⏱️ Session Time
